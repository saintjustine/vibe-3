공공직군 행정업무 슈퍼앱 문서 종합

팀 일정 공유, 엑셀 업무 자동화, 민원 대응 챗봇, 공공행정 뉴스 수집을 하나의 웹 애플리케이션으로 구현하기 위한 요구사항, 아키텍처, 운영 기준을 요약한 문서입니다.

PRD.md Architecture.md Operation.md

제품 목표

PRD
팀원 스케줄 관리

휴가, 근무, 출장, 교육 일정을 캘린더에서 공유하고 팀원별, 유형별로 필터링합니다.

엑셀 업무 자동화

특정 컬럼 기준 파일 분할, 여러 파일 병합, 결과 다운로드를 지원합니다.

민원 대응 챗봇

민원 매뉴얼을 근거로 대응 방향, 확인 사항, 응대 스크립트 초안을 제공합니다.

뉴스 기사 수집

공공행정 관련 키워드 뉴스를 매일 아침 수집하고 중복을 제거합니다.

운영 기준

Operation
항목 기준
npm PowerShell에서는 `npm.cmd` 사용 권장
uv `--cache-dir ..\.uv-cache` 사용 권장
Python `uv run python`으로 실행
SQLite CLI 없이 Python 내장 sqlite3 사용 가능

실행 명령

Commands

Frontend

cd C:\Users\admin\Desktop\DAY3_RPA\frontend
npm.cmd install
npm.cmd run dev

Backend

cd C:\Users\admin\Desktop\DAY3_RPA\backend
uv --cache-dir ..\.uv-cache sync
uv --cache-dir ..\.uv-cache run uvicorn app.main:app --reload

현재는 기능 소스 코드가 아직 없으므로 `npm.cmd run dev`와 `uvicorn app.main:app` 실행은 구현 이후 기준입니다.

주요 리스크와 대응

Risk
리스크 영향 대응
민원 매뉴얼 품질 부족 챗봇 답변 품질 저하 근거 표시, 근거 부족 시 답변 제한
엑셀 형식 다양성 처리 실패 증가 컬럼 검증, 미리보기, 명확한 오류 메시지
뉴스 수집 약관 저작권 또는 접근 제한 RSS/API 우선, 본문 전문 저장 제한
SQLite 한계 동시 사용자 증가 시 성능 저하 저장소 계층 분리, PostgreSQL 전환 대비
개인정보 노출 보안 사고 로그 마스킹, 파일 TTL, 권한 관리
